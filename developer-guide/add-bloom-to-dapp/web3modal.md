# Web3Modal


To include `Bloom Wallet` in the list of recommended wallets, please add the following code to your Web3Modal options:

```javascript
createWeb3Modal({
  //...
  featuredWalletIds: [
    '12ffbeb52b37f08bd8224a50e1cd2173286bf432832579861a5480bd662e349d',
  ]
})
```
